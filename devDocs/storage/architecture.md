# SyncManager 架构图

## 系统架构

```
┌─────────────────────────────────────────────────────────────┐
│                    CCGUI Application                        │
└─────────────────────────────────────────────────────────────┘
                              │
                              │ 调用 API
                              ▼
┌─────────────────────────────────────────────────────────────┐
│                       SyncManager                           │
│                                                             │
│  ┌──────────────────────────────────────────────────────┐  │
│  │  performSync()                                        │  │
│  │  - 扫描 Claude 项目                                   │  │
│  │  - 对比 CCGUI 数据                                    │  │
│  │  - 删除过期数据                                       │  │
│  │  - 导入新数据                                         │  │
│  └──────────────────────────────────────────────────────┘  │
│                                                             │
│  ┌──────────────────────────────────────────────────────┐  │
│  │  loadSessionHistoryWithFallback()                     │  │
│  │  - 优先从 CCGUI 加载                                  │  │
│  │  - 如果没有则从 Claude 导入                           │  │
│  │  - 首次导入后缓存到 CCGUI                             │  │
│  └──────────────────────────────────────────────────────┘  │
└─────────────────────────────────────────────────────────────┘
          │                    │                    │
          │                    │                    │
          ▼                    ▼                    ▼
┌──────────────┐    ┌──────────────────┐    ┌──────────────┐
│   Project    │    │     Session      │    │   History    │
│   Config     │    │     Config       │    │   Manager    │
│   Manager    │    │     Manager      │    │              │
└──────────────┘    └──────────────────┘    └──────────────┘
          │                    │                    │
          └────────────────────┴────────────────────┘
                              │
                              ▼
                    ┌──────────────────┐
                    │      Logger      │
                    │  (/tmp/ccgui/    │
                    │    app.log)      │
                    └──────────────────┘
```

## 数据同步流程

```
启动时同步:
┌─────────────┐
│ App Ready   │
└──────┬──────┘
       │
       ▼
┌─────────────────────┐
│ performSync()       │
└──────┬──────────────┘
       │
       ├─► 扫描 ~/.claude/projects
       │      └─► 获取项目列表
       │          └─► 获取会话列表
       │
       ├─► 扫描 ~/.ccgui/projects
       │      └─► 获取项目列表
       │          └─► 获取会话列表
       │
       ├─► 对比差异
       │      ├─► 找出过期数据
       │      └─► 找出新数据
       │
       ├─► 删除过期数据
       │      ├─► 删除项目
       │      └─► 删除会话
       │
       ├─► 导入新数据
       │      ├─► 创建项目配置
       │      └─► 创建会话配置
       │
       └─► 返回同步报告
              └─► { projectsAdded, projectsDeleted }
```

## 懒加载历史

```
用户打开会话:
┌─────────────────────┐
│ loadSessionHistory  │
│ WithFallback()      │
└──────┬──────────────┘
       │
       ├─► CCGUI 有历史?
       │      │
       │      ├─► YES ─► 直接加载
       │      │           └─► 返回消息
       │      │
       │      └─► NO
       │            │
       │            ▼
       │       从 Claude 导入
       │            │
       │            ├─► 读取 .jsonl
       │            ├─► 解析消息
       │            ├─► 保存到 CCGUI
       │            └─► 返回消息
       │
       └─► 缓存到内存
```

## 目录结构对比

```
同步前:
~/.claude/projects/
└── -Users-alwaysking-Desktop-myapp/
    ├── session-uuid-1.jsonl  (历史消息)
    ├── session-uuid-2.jsonl
    └── session-uuid-3.jsonl


同步后:
~/.ccgui/projects/
└── -Users-alwaysking-Desktop-myapp/
    ├── project.json                    (项目配置)
    └── sessions/
        ├── session-uuid-1/
        │   ├── session.json            (会话配置)
        │   └── history/
        │       ├── messages.jsonl      (历史消息 - 按需导入)
        │       └── metadata.json       (元数据)
        ├── session-uuid-2/
        │   ├── session.json
        │   └── history/
        │       ├── messages.jsonl
        │       └── metadata.json
        └── session-uuid-3/
            ├── session.json
            └── history/
                ├── messages.jsonl
                └── metadata.json
```

## 同步策略

```
Claude 项目     CCGUI 项目      操作
─────────────────────────────────────────
  存在           不存在      → 导入
  存在           存在        → 保留
  不存在         存在        → 删除
  不存在         不存在      → 无操作
```

## IPC 通信

```
前端 (Vue)              主进程 (Electron)          SyncManager
    │                          │                        │
    ├─► window.electronAPI    │                        │
    │     .syncWithClaude()   │                        │
    │                          │                        │
    │                          ├─► ipcMain.handle     │
    │                          │     ('sync-with-     │
    │                          │      claude')        │
    │                          │                        │
    │                          │                        ├─► performSync()
    │                          │                        │     ├─► 扫描
    │                          │                        │     ├─► 对比
    │                          │                        │     ├─► 删除
    │                          │                        │     └─► 导入
    │                          │                        │
    │                          ├─◄── return result ────┤
    │                          │                        │
    ├─◄── result ─────────────┤                        │
    │                          │                        │
    └─► 更新 UI                │                        │
```

## 错误处理流程

```
操作失败时:
┌─────────────────┐
│ 执行同步操作    │
└────┬────────────┘
     │
     ├─► 成功?
     │      │
     │      ├─► YES ─► 记录 INFO 日志
     │      │           └─► 继续执行
     │      │
     │      └─► NO
     │            │
     │            ▼
     │       记录 ERROR 日志
     │            │
     │            ├─► 捕获异常
     │            ├─► 记录详细信息
     │            └─► 继续处理其他数据
     │                 (不影响整体同步)
     │
     └─► 返回结果
              └─► 包含错误信息
```

## 性能优化

```
启动时:
┌──────────────────┐
│ 只同步元数据     │
│ - 项目配置       │
│ - 会话配置       │
│ - 不加载历史     │
└──────────────────┘
        │
        ▼
   快速启动 ✓


用户打开会话时:
┌──────────────────┐
│ 懒加载历史       │
│ - 优先 CCGUI     │
│ - 否则导入       │
│ - 缓存到本地     │
└──────────────────┘
        │
        ▼
   按需加载 ✓
```
